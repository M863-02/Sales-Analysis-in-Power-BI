# Sales Analysis in Power BI
# Overview
This project presents an interactive Sales Analysis Dashboard built using Microsoft Power BI.
The dashboard provides a comprehensive view of sales performance, profitability, customer behavior, and regional trends using historical sales data stored in CSV format.
The objective of this project is to transform raw sales data into actionable business insights through effective data modeling, DAX calculations, and visually intuitive reports.
# Objective
- Analyze overall sales and profit performance  
- Track sales trends across years and months  
- Understand category-wise and segment-wise sales contribution  
- Identify top-performing countries and customers  
- Enable data-driven decision-making through interactivity  
# Tools and Technologies
- PowerBI Desktop
- DAX
- Power Query
#  Data Preparation & Modeling
- Imported CSV data into Power BI
- Cleaned and transformed data using Power Query
- Created relationships between tables
- Built calculated measures using DAX, including:
- Total Sales
- Total Profit
- Total Orders
- Customer Count
- Year-wise and Month-wise aggregations

- Technology contributes the highest share of total sales
- Consumer segment generates the maximum profit
- Sales show a consistent upward trend, peaking in 2014
- Certain months (e.g., August and September) record higher sales
- A small number of countries contribute the majority of revenue
